## JavaScript 获取当前时间戳：

__第一种方法：__

	var timestamp = Date.parse(new Date());
	// 结果：1280977330000

__第二种方法：__

	var timestamp = (new Date()).valueOf();
	// 结果：1280977330748

__第三种方法：__

	var timestamp=new Date().getTime()；
	// 结果：1280977330748

	第一种：获取的时间戳是把毫秒改成000显示，
	第二种和第三种是获取了当前毫秒的时间戳。
